# Bug Predictor

## Description
AI-powered tool to predict risk-prone files in software projects using the NASA Promise dataset.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python backend/app.py
   ```

## Features
- **Admin**: Train models using NASA Promise datasets.
- **User**: Upload code files to predict bug risk.
